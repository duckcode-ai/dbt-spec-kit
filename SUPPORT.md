# Support

## Usage questions

Use GitHub Discussions if enabled, or open a GitHub issue with the `question` label. Include the dbt
adapter, warehouse preset, and the command you ran.

## Bugs

Use the bug report issue template. Include a minimal reproduction and remove private warehouse names,
credentials, and customer data.

## Feature requests

Use the feature request template for new validators, warehouse presets, skills, or examples.
